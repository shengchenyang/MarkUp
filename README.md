# MarkUp
Markdown Parser Software.

## 前言

此项目的目的是介绍使用 `python` 快速开发 `GUI` 的方法；
不使用 `pyqt`, `pyside` 之类的工具，而是使用 `pywebview` 开发和转化已有 `web` 项目来达到高效的目的。
